# Mrs G Amazon Page

Hi! This little project lets you, in a fun way, explore what a software engineer does on a pretend Amazon Your Account page.

## How To Get It Running
1. **Install GitHub Desktop (works on Mac + Windows)**
   - Go to https://desktop.github.com and click the purple download button.
   - Open the downloaded file, keep clicking "Next"/"Continue" until it finishes.
2. **Grab this project with GitHub Desktop**
   - Open GitHub Desktop, click "File" ➜ "Clone repository." ➜ "Url"
   - Into Repository URL, Paste in `https://github.com/WarrenFerrell/Mrs_G_amazon_account_page.git`
   - Pick a folder you like (Desktop is fine) and click "Clone."
3. **Open the page**
   - In GitHub Desktop, press "Show in Finder" (Mac) or "Show in Explorer" (Windows).
   - Double-click `Your Account.html`. Your browser shows the page right away.
   - Close the tab when you are done exploring.

## Stay Safe While Exploring
- If you or anyone nearby is susceptible to photosensitive epilepsy, do not enable crazy speed.
